# Everyday Events Bot

`Everyday Events Bot` - это Telegram-бот, который превращает повседневные задачи в настраиваемую игру.

## Что умеет

- Пользователь сам создает любое количество своих задач.
- Для каждой задачи можно задать очки и таймер по умолчанию.
- Можно запускать задачу с таймером, своим таймером или без таймера.
- За завершение задач начисляются очки.
- Есть награды за достижение: например, дойти до 100 очков и забрать награду.
- Есть магазин отдыха: например, потратить 200 очков на сериал, прогулку, десерт или любую свою награду.
- Есть кланы с кодом вступления, таблицей участников и лентой событий.
- Все хранится локально в `SQLite`, поэтому после перезапуска данные не теряются.

## Стек

- Python 3.9+
- `aiogram` для Telegram-бота
- `SQLite` для хранения данных
- `APScheduler` для напоминаний по таймерам

## Структура

- `main.py`
- `bot/config.py`
- `bot/database.py`
- `bot/handlers.py`
- `bot/scheduler.py`

## Запуск

1. Создайте бота через `@BotFather` и получите токен.
2. Вставьте токен в `.env` или создайте `bot_token.txt` по примеру `bot_token.example.txt`.
3. Перейдите в папку проекта и запустите бота из терминала:

```bash
cd /path/to/everyday-events-bot
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 main.py
```

Файл `run-local.command` - это обычный bash-скрипт для терминала. Запускайте его так:

```bash
bash ./run-local.command
```

## Основные разделы бота

- `Мои задачи` - создание, запуск и удаление задач.
- `Активная задача` - быстрое завершение или отмена текущей задачи.
- `Магазин и награды` - достижения и магазин отдыха.
- `Кланы` - создание клана, вступление по коду, рейтинг и события.
- `Настройки` - очки по умолчанию и массовое обновление ценности задач.
- `Статистика` - личный прогресс.

## Как хранятся данные

База данных создается автоматически в `data/everyday_events.db`.
